``min``
=======

``min`` повертає найменше значення послідовності або набір значень:

.. code-block:: twig

    {{ min(1, 3, 2) }}
    {{ min([1, 3, 2]) }}

При виклику з відображенням min ігнорує ключі і лише порівнює значення:

.. code-block:: twig

    {{ min({2: "e", 3: "a", 1: "b", 5: "d", 4: "c"}) }}
    {# повертає "a" #}
