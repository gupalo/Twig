``max``
=======

``max`` возвращает наибольшее значение последовательности или набор значений:

.. code-block:: twig

    {{ max(1, 3, 2) }}
    {{ max([1, 3, 2]) }}

При вызове с отображением, max игнорирует ключи и только сравнивает значения:

.. code-block:: twig

    {{ max({2: "e", 1: "a", 3: "b", 5: "d", 4: "c"}) }}
    {# возвращает "e" #}
