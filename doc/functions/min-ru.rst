``min``
=======

``min`` возвращает наименьшее значение последовательности или набор значений:

.. code-block:: twig

    {{ min(1, 3, 2) }}
    {{ min([1, 3, 2]) }}

При вызове с отображением, min игнорирует ключи и только сравнивает значения:

.. code-block:: twig

    {{ min({2: "e", 3: "a", 1: "b", 5: "d", 4: "c"}) }}
    {# возвращает "a" #}
