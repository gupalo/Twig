``max``
=======

``max`` повертає найбільше значення послідовності або набір значень:

.. code-block:: twig

    {{ max(1, 3, 2) }}
    {{ max([1, 3, 2]) }}

При виклику з відображенням max ігнорує ключі і лише порівнює значення:

.. code-block:: twig

    {{ max({2: "e", 1: "a", 3: "b", 5: "d", 4: "c"}) }}
    {# returns "e" #}
