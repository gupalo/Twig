``verbatim``
============

Тег ``verbatim`` обозначает разделы как необработанный текст, который не следует 
анализировать. Например, чтобы добавить синтаксис Twig в качестве примера в шаблон, вы можете использовать
этот фрагмент:

.. code-block:: html+twig

    {% verbatim %}
        <ul>
        {% for item in seq %}
            <li>{{ item }}</li>
        {% endfor %}
        </ul>
    {% endverbatim %}
